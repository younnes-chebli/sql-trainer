Test Code Editor

Ce composant permet d'illustrer la manière d'utiliser le composant code-editor pour éditer du code SQL, et plus particulièrement comment on peut faire un binding bidirectionnel de sa valeur avec un attribut (ici query) du composant parent. Pour cela, on a ajouté un textarea qui fait également un binding bidirectionnel vers le même attribut.

Il illustre aussi à travers la checkbox "Read Only" la manière dont on peut contrôler le fait qu'il soit modifiable ou en lecture seule.
